Backup e Restauração
Informações sobre Backup
  • Backup Completo: Exporta todos os dados do sistema (usuários, livros, empréstimos e escolas)
  • Backups Específicos: Exporta apenas uma categoria de dados
  • Formato: Arquivos JSON compatíveis com o sistema
  • Restauração: Substitui todos os dados existentes pelos do backup
⚠️ ATENÇÃO: Exporta todos os dados e APAGA TUDO do Firestore
Atenção - Restauração de Backup

A restauração de backup irá sobrescrever todos os dados existentes. Certifique-se de fazer um backup atual antes de restaurar dados antigos.
